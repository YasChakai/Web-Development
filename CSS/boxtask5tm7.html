Als je ruimte wil maken tussen de border en content gebruik je de Padding

Als je ruimte wiltmaken tussen de border en buiten de box gebruik je Margin

Demonstratie Box model

Het CSS-boxmodel is in wezen een box die om elk HTML-element loopt. Het bestaat uit: randen, opvulling, marges en de daadwerkelijke inhoud

De box heeft een breedte van 130px, Hoogte van 130px.

Met width kan je de breedte aanpassen & met height de hoogte
